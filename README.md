## Installation

To install `picoprobe_portal`:
```bash
git clone https://github.com/ramanathanlab/picoprobe-portal.git
cd picoprobe-portal
python -m venv env
source env/bin/activate
make install
```

## Initial Setup
```bash
django-admin startproject picoprobe_portal
```
